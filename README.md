In this game, the bird must continuously jump to avoid oncoming pipes. Each pipe the bird successfully passes earns 1 point.

## How to Play

Start the game: Click the central "Tap" area.

Jump: Press Space to make the bird flap upward.
If no key is pressed, the bird will naturally fall due to gravity.

Objective: Navigate through the gaps between pipes.
The gap size decreases over time, increasing the difficulty.

## Game Controls

Pause: Press Esc to pause the game.
While paused: Click "Resume" to continue playing. Click "Menu" to restart.
Exit: Click the "Exit" button in the upper-right corner at any time.
